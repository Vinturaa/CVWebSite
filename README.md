# CVWebSite

A desktop-style portfolio website showcasing my work, skills, and experience through an interactive, desktop-inspired user interface.

## Overview

This project simulates a desktop environment in the browser, presenting portfolio content (about me, projects, skills, contact) as interactive windows/icons rather than a traditional scrolling page.

## Features

- Desktop-style UI/UX
- Interactive windows for different sections (About, Projects, Skills, Contact)
- Responsive design
- Custom icons and taskbar-like navigation

## Tech Stack

- HTML
- CSS
- JavaScript

## Getting Started

Clone the repo:

```bash
git clone https://github.com/Vinturaa/CVWebSite.git
cd CVWebSite
```

Open `index.html` in your browser, or use a local server (e.g. VS Code Live Server extension).
